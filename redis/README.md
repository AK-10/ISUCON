## HOW TO USE THIS template

### ファイルの取得

```bash
wget https://raw.githubusercontent.com/naoto67/ISUCON/master/redis/template.go -O cache.go
```

### env.shの使用

```bash
wget https://raw.githubusercontent.com/naoto67/ISUCON/master/redis/env.sh
# host, portを書き換える
source env.sh # 環境変数
```
